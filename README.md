# <center> 健康管理APP

页面包含了各种元素、方法使用。

**主要包括以下功能模块：**

1. 首页（轮播图、横向滚动菜单等）
2. 发现
3. 购物车
4. 我的
5. 登录/注册
6. 国家地区选择菜单（带导航）
7. 接口请求
8. 页面跳转
